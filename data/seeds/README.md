## Regras para os seeds

Quando os seeds forem criados, deverão respeitar:

1. **Sem dados inventados.** Todo dado deve ter origem real/confiável.
2. **Preço e imagem como pendência** quando não consolidados — nunca preencher
   com valores fictícios.
3. **Sem validação oficial não comprovada** (AGROFIT/MAPA).
4. Formato e estrutura alinhados ao [DER](../../docs/04-modelagem-banco-der.md) e
   ao [dicionário de dados](../../docs/05-dicionario-de-dados.md).
